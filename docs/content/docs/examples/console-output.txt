Usage
  $ mwap <command> [options]

Available Commands
  build     Build your project for production.
  dev       Run your project in dev mode.
  export    Static export your app.
  start     Run your build.

For more info, run any command with the `--help` flag
  $ mwap build --help
  $ mwap dev --help

Options
  -v, --version    Displays current version
  -h, --help       Displays this message
