# Custom App

MWAP uses the `App` component to wrap your root pages. You can override it by creating either an `app.tsx`, `app.ts`, `app.js`, or `app.jsx`.

Overriding the App allows you to do things like:

- Add global styles
- Render an app shell
- Provide global state
- And even `useLoader`
